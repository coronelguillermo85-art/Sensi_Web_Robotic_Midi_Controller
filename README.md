<h1>🎹 SENSI WEB</h1>

<h3>Secuenciador MIDI Heurístico y Entorno de Composición Generativa</h3>

<p>
Aplicación musical experimental basada en navegador que combina
secuenciación MIDI, generación algorítmica, improvisación asistida y
herramientas de exploración sonora en tiempo real.
</p>

</div>

<hr>

<h2>🚀 Características</h2>

<ul>
<li>🎹 Piano Roll interactivo</li>
<li>🎼 Generación automática de secuencias musicales</li>
<li>⚡ Modo de improvisación heurística</li>
<li>🔄 Evolución y mutación de patrones</li>
<li>🎵 Importación de archivos MIDI</li>
<li>🎚 Controles de densidad, armonía, caos y evolución</li>
<li>🎹 Compatibilidad con controladores MIDI mediante Web MIDI</li>
<li>🔊 Síntesis integrada mediante Web Audio API</li>
<li>📊 Visualización en tiempo real</li>
<li>🤖 Interfaz inspirada en asistentes musicales inteligentes</li>
<li>🌐 Funciona completamente desde el navegador</li>
<li>🚀 Despliegue estático en Vercel, Netlify o GitHub Pages</li>
</ul>

cd sensi-web
</pre>

<p>

</p>

<pre>

</pre>

<p>

</p>

<hr>

<h2>🎹 Uso</h2>

<ol>
<li>Abrir la aplicación en Google Chrome, Edge u Opera.</li>

<li>Conectar un controlador MIDI compatible (opcional).</li>

<li>Dibujar notas en el Piano Roll.</li>

<li>Utilizar <b>IMPROVISA</b> para generar secuencias automáticamente.</li>

<li>Aplicar <b>EVOLUCIONAR</b> para modificar patrones existentes.</li>

<li>Importar archivos MIDI para crear acompañamientos.</li>
</ol>
